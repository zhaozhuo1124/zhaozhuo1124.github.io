术语

发表于 2020-12-12 更新于 2020-12-14 分类于 Python学习笔记 阅读次数： Valine：
本文记录了一些有关python的注意事项，同时一些术语也适用于其他编程语言。

注意

  • Python是一个区分大小写的编程软件
  • pen.speed() 的最大速度为 0
  • 函数包含方法（functions contain methods）

术语

  • 程序

    面向计算机的一组指令

    由代码行组成

    每行告诉计算机关于这些指令的一个特定细节信息

  • 变量（variable）

    将代码中的名称与计算机内存中的某些数据关联起来

  • 赋值（assignment）

  • 字符串/串（string）

    由数字、字母、下划线组成的一串字符

    是编程语言中表示文本的数据类型

    example：“yellow”

  • 整数

    一个没有分数或小数部分的数

    可以是正数、负数或零

    -5 、10、 0

  • 列表

    放入方括号内，并用逗号分隔各项

    [1,2,3]、["hello","yelloew","stuff"]

  • 模块（module）

    一个包含实用代码的文件

    可以 import 到程序中

    a file that has a collection of useful code that can be used in other Python programs

  • 方法（method）

    一个有名称的代码块

    可以调用它来使turtle执行某个操作

    • 方法调用（method calls）

  • 注释

    针对人类读者的信息

    计算机读取代码时会忽略注释

    在Python中注释行以 # 开头

    快捷键可用 Ctrl + /

  • 错误

    • 语法错误

      • （）前后不对应
      • for循环忘记加 :

    • 用法错误

      要求计算机执行不合理的操作

      常见提示：

      NameError

      TypeError

      ZeroDivisionError

    • 逻辑错误

      计算机行为与编程者想要的不一致

  • 变量

    变量作用域是定义函数所在的程序文本

    • 局部变量

      在函数内定义的变量

      只能由该函数内的代码引用

    • 全局变量

      在任何函数外的变量
